#  Tetris

A simple clone of [Tetris](https://en.wikipedia.org/wiki/Tetris)

## control

- <kbd>↓</kbd> soft drop
- <kbd>→</kbd> right
- <kbd>←</kbd> left
- <kbd>Space</kbd> hard drop
- <kbd>r</kbd> rotate clock-wise
- <kbd>q</kbd> anti clock-wise
